# Vercel ダッシュボード → Project → Settings → Environment Variables に追加
#
# GAS_WEBAPP_URL … GAS を「ウェブアプリ」としてデプロイしたあとの URL
# 例: https://script.google.com/macros/s/xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx/exec
#
# DISCORD_WEBHOOK_URL …（任意）Discord の「連携サービス」→「ウェブフック」で発行した URL
# 例: https://discord.com/api/webhooks/123456789/abcdefghijklmnopqrstuvwxyz
